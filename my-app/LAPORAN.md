PERTANYAAN REFLEKSI
1. Mengapa Pages Router disebut sebagai routing berbasis file? 
Karena sistem routing di Next.js dibuat otomatis berdasarkan nama dan struktur file di folder pages
2. Apa perbedaan Next.js dengan React standar (CRA)? 
   -Next.js memiliki routing bawaan
   -Mendukung Server-Side Rendering (SSR) dan Static Generation
   -Optimasi performa dan SEO lebih baik
   -Setup lebih lengkap tanpa banyak konfigurasi
   Sedangkan CRA hanya React client-side, routing harus install sendiri dan tidak ada SSR bawaan
3. Apa fungsi perintah npm run dev? 
Untuk menjalankan aplikasi dalam mode development
4. Apa perbedaan npm run dev dan run build ?
npm run dev	Menjalankan project untuk development
npm run build Membuat versi produksi yang sudah dioptimasi
